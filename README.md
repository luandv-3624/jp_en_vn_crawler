# Dictionary API (JP → EN / JP → VI)

This project provides a **dictionary lookup API** used for a reading application.  
When a user taps a Japanese word while reading a novel, the system returns its meaning in:

- **Japanese → English (JP → EN)**
- **Japanese → Vietnamese (JP → VI)**

The API is implemented using **FastAPI** and runs inside **Docker**.

---

# Architecture

Frontend sends a lookup request to the API.

```
Frontend
   ↓
FastAPI Dictionary API
   ↓
Local Dictionary Database
```

Data generation pipeline:

```
JMdict XML
   ↓
Parser
   ↓
Database (JP → EN)

JMdict vocabulary
   ↓
Crawler (JDict API)
   ↓
Database (JP → VI)
```

# Data Pipeline

The dictionary data must be generated before running the API.

---

# 1. Parse JMdict (JP → EN)

The English dictionary data comes from **JMdict**, a widely used open Japanese-English dictionary dataset.

Steps:

1. Download the JMdict dataset.
2. Run the parsing script.

```bash
python scripts/import_jmdict.py
```

The script will:

- Parse the JMdict XML file
- Extract:
  - Kanji
  - Reading
  - English meanings
- Insert the data into the database

Estimated dataset size:

```
~200,000 entries
```

---

# 2. Crawl Japanese → Vietnamese

There is no large open dataset for **JP → VI**, so Vietnamese meanings are generated using a crawler.

The crawler calls the **JDict API**.

Example request:

```
https://jdict.net/search?keyword=こんにちは&type=word
```

Run the crawler:

```bash
python scripts/import_vi_from_jdict.py
```

Crawler workflow:

```
Select entries without Vietnamese meaning
        ↓
Call JDict API
        ↓
Parse response
        ↓
Save Vietnamese meanings to database
```

The crawler runs in **batch mode** to avoid rate limits.

Note:

- The crawler is only used to **generate the dataset**
- Runtime API queries **local database only**

# Example API

Search for a word:

```
GET /dictionary/search?word=食べる
```

Example response:

```json
{
  "word": "食べる",
  "reading": "たべる",
  "meanings_en": [
    "to eat"
  ],
  "meanings_vi": [
    "ăn"
  ]
}
```
